A premium, scroll-aware navigation header that dynamically adjusts its layout, logo scale, and background density as the user scrolls. Part of the Premium UI series.

<ComponentPreview name="morphing-navbar" />

## Installation

### CLI

```bash
npx react-ui-component add morphing-navbar
```

### Manual

1. Copy the code from the component tab.
2. Create a file at `components/ui/morphing-navbar.tsx`.
3. Paste the code.

## Usage

```tsx
import { MorphingNavbar } from "@/components/ui/morphing-navbar";
import { GraduationCap } from "lucide-react";

export default function Example() {
  return (
    <MorphingNavbar 
      title="Morphing Navbar"
      logo={GraduationCap}
      breadcrumbSteps={[
        { label: "Components" },
        { label: "Navigation" }
      ]}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `title` | `string` | `"Morphing Navbar"` | The title text displayed next to the logo. |
| `logo` | `ElementType` | `GraduationCap` | The icon component to display as the logo. |
| `breadcrumbSteps` | `Step[]` | `[]` | Array of breadcrumb steps to display in expanded mode. |
| `scrollThreshold` | `number` | `36` | Scroll distance in pixels before morphing to collapsed state. |
| `targetId` | `string` | - | Optional ID of a scrollable container to track instead of the window. |
