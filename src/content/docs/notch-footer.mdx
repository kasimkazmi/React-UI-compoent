A premium, animated footer and call-to-action banner featuring a mathematically perfect curved notch pocket. Built with **Framer Motion** and **Tailwind CSS**. It is fully responsive and supports custom branding, dot grid indicators, and physics-based hover scaling out of the box.

<ComponentPreview name="notch-footer" />

## Installation

### CLI

```bash
npx shadcn@latest add http://localhost:3000/registry/notch-footer.json
```

### Manual

1. Copy the code from the component tab.
2. Create a file at `components/notch-footer.tsx`.
3. Paste the code.

## Usage

```tsx
import { NotchFooter } from "@/components/notch-footer";

export default function Example() {
  const handleAction = () => {
    console.log("Action clicked!");
  };

  return (
    <NotchFooter
      title="Stop guessing about your digital experience with LogRocket"
      buttonText="Get started for free"
      onButtonClick={handleAction}
      bgColorClass="bg-gradient-to-r from-violet-600 via-indigo-600 to-purple-700"
      notchFillColor="#FAF9F7" // Must match parent background color
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `title` | `string` | `"Stop guessing..."` | The primary headline rendered inside the banner |
| `buttonText` | `string` | `"Get started for free"` | Text inside the bottom outline action button |
| `onButtonClick` | `() => void` | - | Callback function triggered when the CTA button is clicked |
| `buttonHref` | `string` | - | Optional destination link for the button (turns it into an `a` tag) |
| `logo` | `ReactNode` | - | Custom logo element floating gracefully inside the notch pocket |
| `bgColorClass` | `string` | `bg-gradient-to-r...` | Tailwind class(es) setting the main banner's background |
| `notchFillColor` | `string` | `"#F7F5F3"` | Hex color filling the notch overlay (should match your page background) |
| `className` | `string` | - | Additional CSS styling classes for the container |
